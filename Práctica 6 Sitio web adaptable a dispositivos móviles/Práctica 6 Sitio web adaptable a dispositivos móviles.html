📝 index.html (plantilla base)

Este código debe repetirse en productos.html y contacto.html, cambiando solo el contenido central.

<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <!-- ETIQUETA PRIMORDIAL para adaptabilidad móvil --> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Artesanías "Don Pepe" - Responsivo</title> <link rel="stylesheet" href="reglas.css"> </head> <body> <header class="header"> <h1>Poniendo en Línea mis Raíces</h1> </header> <nav class="navbar"> <ul class="menu"> <li><a href="index.html">Inicio</a></li> <li><a href="productos.html">Productos</a></li> <li><a href="contacto.html">Contacto</a></li> </ul> </nav> <main class="contenedor-principal"> <section class="contenido"> <h2>Bienvenido a nuestro taller</h2> <p>Productos elaborados a mano con identidad regional.</p> <!-- Elementos multimedia (ejemplo de 2 imágenes) --> <img src="img/artesania1.jpg" alt="Producto 1" class="img-fluida"> <img src="img/artesania2.jpg" alt="Producto 2" class="img-fluida"> </section> <aside class="lateral"> <h3>Noticias</h3> <p>Nuevos diseños disponibles este mes.</p> </aside> </main> <footer class="pie"> <p>&copy; 2026 Emprendimiento Local</p> </footer> </body> </html>

🔁 Recuerda: en productos.html y contacto.html cambia el contenido de .contenido y .lateral para que sean diferentes.

🎨 reglas.css (Mobile First + Media Queries)

/* ===== ESTILOS GENERALES (BASE MÓVIL) ===== */ * { box-sizing: border-box; margin: 0; padding: 0; } body { font-family: Arial, sans-serif; background-color: #f4f4f4; } /* Encabezado y pie */ .header, .pie { background-color: #333; color: white; text-align: center; padding: 20px; } /* Barra de navegación */ .navbar { background-color: #800020; } .menu { list-style: none; display: flex; flex-wrap: wrap; justify-content: center; } .menu li a { color: white; text-decoration: none; padding: 15px; display: block; } .menu li a:hover { background-color: #a04040; transition: 0.3s; } /* Contenedor principal FLEXIBLE */ .contenedor-principal { display: flex; flex-wrap: wrap; /* EN MÓVIL: SE APILAN */ padding: 10px; } .contenido { flex: 1 1 100%; /* OCUPA 100% EN MÓVIL */ padding: 10px; background: white; } .lateral { flex: 1 1 100%; /* OCUPA 100% EN MÓVIL */ padding: 10px; background: #ddd; } /* Imágenes fluidas (nunca se desbordan) */ .img-fluida { max-width: 100%; height: auto; border-radius: 8px; margin: 10px 0; } /* ===== MEDIA QUERY PARA TABLETS Y ESCRITORIO (pantallas > 700px) ===== */ @media (min-width: 700px) { .contenedor-principal { flex-wrap: nowrap; /* ESCRITORIO: UNO AL LADO DEL OTRO */ } .contenido { flex: 3; /* Ocupa 3/4 del espacio */ margin-right: 10px; } .lateral { flex: 1; /* Ocupa 1/4 del espacio */ } } /* Ajustes adicionales para pantallas muy grandes */ @media (min-width: 1200px) { .contenedor-principal { max-width: 1200px; margin: 0 auto; } }

✅ viewport, flex-wrap y @media son los protagonistas.

🖼️ Requisito multimedia: al menos 5 elementos

Distribúyelos entre las tres páginas. Ejemplo:

📸 2 imágenes en inicio 📸 2 imágenes en productos 🎥 1 video en contacto 🔊 (opcional) audio

Importante: Usa siempre la clase img-fluida en las imágenes para que se ajusten al contenedor.

🔑 Puntos clave para el éxito

  1. MetaTag Viewport: Sin él, el sitio no es responsivo. Está en el <head>.
  2. Flexbox + flex-wrap: En móvil, flex-wrap: wrap hace que los elementos se apilen. En escritorio, nowrap los coloca en fila.
  3. Media Query: La regla @media (min-width: 700px) cambia la disposición para pantallas grandes.
  4. Pruebas: Redimensiona el navegador o usa herramientas de desarrollador (F12) para simular móvil.
  5. Tres páginas vinculadas: Verifica que los enlaces del menú funcionen entre index.html, productos.html y contacto.html.

🧠 Tips para el éxito responsivo

  • 📱 Prueba en modo dispositivo: en Chrome, F12 → ícono de móvil.
  • 🖼️ Todas las imágenes deben tener max-width: 100% para que nunca sean más anchas que la pantalla.
  • 🔗 Antes de entregar, verifica que los enlaces del menú realmente abran las otras páginas.
  • 📂 Asegúrate de que la carpeta img/ contenga las imágenes y estén bien nombradas.
